# Spidercarp Landing

https://github.com/user-attachments/assets/7fb7daeb-882e-4060-b3e7-88ddd048534c

Landing page para el streamer Spidercarp.

## Tecnologías

- Svelte 5
- Vite 8
- TypeScript

## Estructura

- **Inicio:** Sección superior de presentación.
- **Stream:** Reproductor de Kick en vivo y clips de video.
- **Redes Sociales:** Enlaces de perfiles y publicaciones de Twitter.
- **Colaboraciones:** Sección de colaboraciones, y reproductor de YouTube.
- **Comunidad:** Enlaces de acceso a Discord y WhatsApp.

## Replicación

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Compilar producción
npm run build
```
